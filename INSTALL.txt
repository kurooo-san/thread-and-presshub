Thread and Press Hub - Apparel Studio Ordering System
========================================

INSTALLATION INSTRUCTIONS
=========================

REQUIREMENTS:
- PHP 7.4 or higher
- MySQL/MariaDB
- Apache with mod_rewrite
- XAMPP (or similar development environment)

STEP 1: DATABASE SETUP
======================

1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Click "New" to create a database
3. Enter database name: threadpresshub
4. Select database
5. Go to "Import" tab
6. Choose threadpresshub.sql file
7. Click "Go" to import

STEP 2: VERIFY FILES
====================

Check that these files exist in C:\xampp\htdocs\\

Files to verify:
- index.php
- login.php
- register.php
- shop.php
- cart.php
- checkout.php
- orders.php
- profile.php
- logout.php
- order_confirmation.php
- payment_gcash.php
- order_details.php
- admin/dashboard.php
- admin/products.php
- admin/orders.php
- admin/users.php
- admin/order_details.php
- includes/config.php
- includes/header.php
- includes/footer.php
- css/style.css
- threadpresshub.sql
- README.md
- SETUP_GUIDE.md

Folders to verify/create:
- images/
- images/products/
- includes/
- admin/
- css/

STEP 3: FILE PERMISSIONS
=========================

Windows:
1. Right-click threadpresshub folder
2. Properties → Security
3. Click Edit
4. Select your user
5. Check "Modify"
6. Apply → OK

STEP 4: DATABASE CONFIGURATION
==============================

The includes/config.php contains:
- Host: localhost
- User: root
- Password: (empty by default)
- Database: threadpresshub

If your MySQL has different credentials, edit includes/config.php

STEP 5: START SERVICES
======================

1. Open XAMPP Control Panel
2. Click "Start" for Apache
3. Click "Start" for MySQL
4. Wait for both to show green

STEP 6: FIRST ACCESS
====================

1. Open browser
2. Visit: http://localhost/threadpresshub/
3. You should see the Thread and Press Hub homepage

STEP 7: ADMIN LOGIN
===================

First admin account:
- Email: admin@threadpresshub.com
- Password: admin123

IMPORTANT: Change this password after first login!

To change admin password:
1. Login with admin account
2. Click Profile (top right)
3. Enter current password
4. Enter new password
5. Save Changes

STEP 8: ADD PRODUCTS
====================

1. Login as admin
2. Click Admin (top navigation)
3. Click "Manage Products"
4. Fill in product details:
   - Name
   - Price
   - Category
   - Description
   - Image (upload JPG/PNG)
5. Click "Add Product"

STEP 9: TEST THE SYSTEM
=======================

Create a test customer account:
1. Go to http://localhost/threadpresshub/
2. Click "Register"
3. Fill in details
4. Select account type (Regular, PWD, or Senior)
5. Click "Create Account"

Test ordering:
1. Click "Login" or go to menu
2. Browse products
3. Add to cart
4. View cart
5. Checkout
6. Select payment method
7. Enter delivery address
8. Place order

TROUBLESHOOTING
===============

ERROR: "Connection failed"
→ Check if MySQL is running
→ Verify database name in includes/config.php
→ Check MySQL user/password

ERROR: "No such file or directory"
→ Verify threadpresshub.sql exists
→ Check all PHP files are in correct folders
→ Check includes/config.php database name matches

ERROR: Images not uploading
→ Create images/products/ folder if missing
→ Check folder permissions (must be writable)
→ Check PHP upload size in php.ini

ERROR: Cart not working
→ Check browser allows localStorage
→ Try different browser
→ Clear browser cache

ERROR: Login/register not working
→ Check database has users table
→ Verify config.php database connection
→ Check MySQL service is running

DISCOUNT STRUCTURE
==================

Account Types:
1. Regular: No discount
2. PWD (Person with Disability): 12% discount
   - Requires valid PWD ID
3. Senior Citizen: 15% discount
   - Requires valid Senior ID

Discounts applied at checkout

DELIVERY
========

- Delivery Fee: ₱50
- Free for orders over ₱500 (customizable)
- Delivery address required
- Special instructions supported
