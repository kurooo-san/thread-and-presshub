TROUBLESHOOTING
===============

Problem: Database connection failed
Solution: Check config.php credentials, ensure MySQL is running

Problem: Images not uploading
Solution: Create images/products/ folder, set write permissions

Problem: Cart not working
Solution: Enable browser localStorage, clear cache

Problem: Admin not accessible
Solution: Verify user_type='admin' in database, check session
